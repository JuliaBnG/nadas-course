# The Almighty Tab

The `Tab` key in Julia REPL is very powerful.  It can be used to complete the
code, to type a unicode character, and so on.  For example, you can type `\flf`
and then press `Tab` key, and you will see that the code is completed to
`flfSheep`.  You type `flf` then `Tab` key to get `flfSheep`, then you type
`.g`, and `tab` key again, you will see that the code is completed to
`flfSheep.greet`.

To type a unicode character, you can type `\` followed by the name of the
character, and then press `Tab` key.  For example, you can type `\mu` and then
press `Tab` key, and you will see that the code is completed to `μ`.

An emoji start with `\:`, then its name.  For example, `\:sheep:` and then
press `Tab` key, and you will see that the code is completed to `🐑`.

`\^2` then `Tab` key gives you `²`.

`\sqrt` then `Tab` key gives you `√`.

If you know some $\LaTeX$, you will find it easy to type math symbols.
