# The Almighty Tab

The `Tab` key in the Julia REPL is extremely versatile. It is used for code 
completion, typing Unicode characters, and more. For example, typing `\flf` 
and pressing the `Tab` key will complete the text to `flfSheep`. If you type 
`flf`, press `Tab` to get `flfSheep`, then type `.g` and press `Tab` again, 
it will complete to `flfSheep.greet`.

To type a Unicode character, type `\` followed by the name of the character, 
then press the `Tab` key. For example, typing `\mu` and pressing `Tab` 
will result in `μ`.

Emojis start with `\:` followed by their name. For example, typing `\:sheep:` 
and pressing `Tab` will produce `🐑`.

Typing `\^2` then `Tab` gives you `²`.

Typing `\sqrt` then `Tab` gives you `√`.

Familiarity with $\LaTeX$ makes typing mathematical symbols even easier.
