Note if you want to register your package in the Julia General
Registry, some tweaks are needed.  That is beyond the scope of this
course.  With the help of AI, e.g., `GitHub Copilot`, that is not a
big deal.

There is also a new file `Manifest.toml` in the project directory.

```bash
echo Manifest.toml >> .gitignore
git add .gitignore
```

Above means that we will not track `Manifest.toml` in `git`.  The
reason is that `Manifest.toml` is automatically generated by `Julia`.
It can change very much from time to time. It is thus excluded from
version control.
